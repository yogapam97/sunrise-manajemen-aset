Once you have created an operation group, you can perform bulk operations on selected assets:

2. Select the operation group you created.
3. Click on **"Do Operation"** to start the bulk operation process.
4. Choose the fixed assets you want to include in the operation.
5. Configure the specific changes required for each task (e.g., new locations for relocations, new lifecycle stages for transitions).
6. Save the operation to apply the changes.
