# Stageholder Fixed Asset Management Rest API

Feature:

> Fixed Asset Management

> Depreciation Calculation

> Assessment Report

> KPI/Goals Tracking

> Designed for Enterprise Management

## Copy Environment Variable

```
cp .env.example .env
```

## Generate Secret Key and Insert into Environment Variable

```
openssl rand -base64 32
```

## Project setup

```
yarn install
```

### Run

```
yarn start
```
